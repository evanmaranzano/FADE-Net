# 基于轻量级混合注意力网络的人脸年龄估计研究

## 摘要

人脸年龄估计是计算机视觉领域的重要研究方向，在智能监控、人机交互、商业分析等领域具有广泛应用。然而，现有方法往往依赖深层网络和大量参数，难以部署于资源受限的边缘设备。针对这一问题，本文提出FADE-Net（Feature-fused Hybrid Attention Distribution Estimation Network），一种面向轻量级场景的人脸年龄估计网络。

FADE-Net以MobileNetV3-Large为骨干网络，提出Pyramid Attention Injection策略，在深层注入Coordinate Attention以增强空间感知能力；设计纹理-语义双流融合架构，显式建模皱纹纹理与脸型结构的双重年龄特征；引入Bottleneck Spatial Pyramid Pooling模块捕获多尺度上下文信息；结合改进的DLDL-v2标签分布学习策略和Mean-Variance Loss，有效处理年龄标签的模糊性问题。

在AFAD数据集上的实验结果表明，FADE-Net在参数量仅为4.84M的情况下，达到MAE 3.057的性能，超越了现有轻量级方法，与参数量超过25M的方法性能相当。消融实验验证了各模块的有效性。本研究为资源受限场景下的人脸年龄估计提供了可行的解决方案。

**关键词**：人脸年龄估计；轻量级网络；注意力机制；标签分布学习；深度学习

## 第一章 绪论

### 1.1 研究背景与意义

人脸年龄估计是指根据人脸图像自动推断出人物年龄的技术。随着深度学习技术的发展，人脸年龄估计在智能监控、身份认证、商业营销、人机交互等场景中的应用价值日益凸显。例如，在智能零售场景中，年龄估计可用于顾客群体分析和精准营销；在社交娱乐应用中，可实现趣味性的年龄预测功能。

然而，实际应用场景往往面临计算资源受限的约束。许多现有方法依赖ResNet-50、VGG-16等深层网络，参数量动辄超过20M，难以部署于移动设备或嵌入式平台。因此，研究轻量级、高效率的人脸年龄估计方法具有重要的理论意义和应用价值。

### 1.2 研究现状

传统人脸年龄估计方法主要依赖于人工设计的特征，如局部二值模式（LBP）、Gabor特征等，但这些特征的表达能力有限，难以捕捉复杂的年龄相关模式。

近年来，深度学习方法在人脸年龄估计任务上取得了显著进展。Rothe等人提出的DEX方法使用VGG-16网络，通过在大规模数据集上预训练，显著提升了估计精度。然而，这类方法普遍存在参数量大、计算成本高的问题。

在轻量化研究方面，MobileNet系列网络通过深度可分离卷积大幅降低了参数量，但直接应用于年龄估计任务时，性能往往不如深层网络。如何在轻量化的同时保持估计精度，是当前研究的重点问题。

### 1.3 本文主要工作

针对上述问题，本文提出FADE-Net轻量级人脸年龄估计网络。主要贡献包括：

1. **提出Pyramid Attention Injection策略**：针对年龄估计任务对空间位置信息敏感的特点，在MobileNetV3深层注入Coordinate Attention，在保持轻量化的同时增强空间感知能力。

2. **设计纹理-语义双流融合架构**：通过分离提取皱纹纹理特征和脸型语义特征，显式建模年龄的双重表征。

3. **构建完整的轻量级训练范式**：集成Bottleneck SPP、DLDL-v2、Mean-Variance Loss等模块，形成针对年龄估计任务的系统性解决方案。

4. **在AFAD数据集上验证方法有效性**：FADE-Net在4.84M参数量下达到MAE 3.057，超越了现有轻量级方法。

### 1.4 论文组织结构

本文共分为六章。第一章介绍研究背景和主要工作；第二章综述相关工作；第三章详细阐述FADE-Net的网络结构和设计思想；第四章介绍实验设置和训练策略；第五章展示实验结果和分析；第六章总结全文并展望未来工作。

## 第二章 相关工作

### 2.1 人脸年龄估计方法

人脸年龄估计方法可分为三类：回归方法、分类方法和标签分布学习方法。

**回归方法**将年龄估计视为回归问题，直接预测年龄数值。这类方法简单直观，但忽略了年龄标签的模糊性——相邻年龄的外貌差异往往难以区分。

**分类方法**将年龄估计视为多分类问题，每个年龄值对应一个类别。这类方法能够学习更丰富的类别间关系，但忽略了年龄的序数性质。

**标签分布学习方法**（Label Distribution Learning, LDL）由Geng等人提出，将单一标签扩展为标签分布，有效处理年龄标签的模糊性问题。Gao等人提出的DLDL方法进一步将其应用于深度学习框架，成为当前主流方法。

### 2.2 轻量级网络架构

轻量级网络设计是移动端部署的关键技术。MobileNet系列通过深度可分离卷积大幅降低计算量；ShuffleNet通过通道混洗增强特征交互；EfficientNet通过神经架构搜索实现精度与效率的平衡。

MobileNetV3结合了NAS搜索和手工设计，在ImageNet分类任务上达到了优秀的精度-效率权衡。然而，直接将分类网络迁移至年龄估计任务往往存在性能瓶颈，需要针对性的架构设计。

### 2.3 注意力机制

注意力机制能够使网络关注关键区域，提升特征表达能力。SE-Net提出的通道注意力通过自适应重标定通道权重提升性能；CBAM同时建模通道注意力和空间注意力；Coordinate Attention将位置信息编码到通道注意力中，更适合需要空间感知的任务。

在人脸年龄估计任务中，眼睛、额头、嘴部等区域的皱纹是重要的年龄线索，空间注意力能够帮助网络聚焦这些关键区域。

### 2.4 多尺度特征融合

多尺度特征融合能够同时捕获局部细节和全局上下文。特征金字塔网络（FPN）通过自顶向下路径融合不同层级的特征；Spatial Pyramid Pooling（SPP）通过多尺度池化获得固定长度的特征表示。

在年龄估计任务中，皱纹纹理需要高分辨率特征捕获，而脸型结构需要低分辨率语义特征表示，多尺度融合具有重要价值。

## 第三章 方法

### 3.1 网络整体架构

FADE-Net的整体架构如图所示，主要由骨干网络、特征融合模块和预测头组成。

```
输入图像 (224×224×3)
    ↓
┌─────────────────────────────────┐
│     MobileNetV3-Large Backbone   │
│  ┌───────────────────────────┐   │
│  │ Blocks 0-5 (浅层，保持SE) │   │
│  └───────────────────────────┘   │
│              ↓                    │
│  ┌───────────────────────────┐   │
│  │ Block 6 → 纹理分支 (40ch) │───┼──→ 纹理特征
│  └───────────────────────────┘   │
│              ↓                    │
│  ┌───────────────────────────┐   │
│  │ Block 12 → 语义分支(112ch)│───┼──→ 语义特征
│  └───────────────────────────┘   │
│              ↓                    │
│  ┌───────────────────────────┐   │
│  │ Blocks 13-15 + CoordAtt   │   │
│  └───────────────────────────┘   │
│              ↓                    │
│     Bottleneck SPP (512ch)       │
└─────────────────────────────────┘
              ↓
    特征融合 (纹理 + 语义 + SPP)
              ↓
        预测头 (1024 → 81)
              ↓
       年龄概率分布
              ↓
       期望值作为预测年龄
```

### 3.2 Pyramid Attention Injection策略

MobileNetV3-Large使用Squeeze-and-Excitation（SE）模块作为通道注意力，但SE模块忽略了空间位置信息。对于年龄估计任务，不同面部区域的年龄线索重要性不同，需要空间感知能力。

本文提出Pyramid Attention Injection策略：**仅在网络深层替换SE模块为Coordinate Attention（CA），浅层保持原始SE模块。**

设计理由：
1. **浅层特征**主要捕获边缘、纹理等底层视觉信息，对空间位置不敏感，SE模块足够且高效。
2. **深层特征**承载语义信息，需要区分眼睛、额头、嘴部等不同区域的重要性，CA模块的空间感知能力至关重要。
3. **只替换后4层**，在性能提升和计算开销之间取得平衡。

CoordAtt模块的实现如下：

$$y_h = \frac{1}{W}\sum_{0\le i < W} x_{:,i}$$

$$y_w = \frac{1}{H}\sum_{0\le j < H} x_{j,:}$$

通过对高度和宽度方向分别进行全局池化，将位置信息编码到注意力权重中。

### 3.3 纹理-语义双流融合架构

年龄的外在表现具有双重性：**纹理特征**（皱纹、斑点）反映皮肤老化程度，**语义特征**（脸型、五官比例）反映骨骼发育和脂肪分布。两类特征的空间分辨率需求不同。

本文设计双流融合架构：

**纹理分支**：取自Block 6输出，空间分辨率28×28，通道数40。高分辨率特征能够精细捕捉皱纹纹理。

**语义分支**：取自Block 12输出，空间分辨率14×14，通道数112。低分辨率特征编码全局脸型结构。

融合过程：
1. 两分支分别通过1×1卷积投影至64通道
2. 纹理分支下采样至14×14与语义分支对齐
3. 通过可学习权重进行注意力加权融合：
   $$f_{fused} = softmax(w_0) \cdot f_{texture} + softmax(w_1) \cdot f_{semantic}$$
4. 融合特征投影至128维向量

### 3.4 Bottleneck Spatial Pyramid Pooling

传统全局平均池化（GAP）丢失空间信息，本文引入Bottleneck SPP模块：

$$p_1 = x$$
$$p_2 = MaxPool_{5\times5}(x)$$
$$p_3 = MaxPool_{9\times9}(x)$$
$$p_4 = MaxPool_{13\times13}(x)$$
$$out = Conv_{1\times1}(Concat(p_1, p_2, p_3, p_4))$$

通过多尺度最大池化捕获不同感受野的空间信息，最后通过1×1卷积压缩通道数至512，保持轻量化。

### 3.5 标签分布学习

年龄标签存在固有模糊性——相邻年龄的外貌高度相似。标签分布学习将单点标签扩展为概率分布：

$$P(j|x) = \frac{1}{Z}\exp\left(-\frac{(j-y)^2}{2\sigma^2}\right)$$

其中$y$为真实年龄，$j$为标签空间中的年龄值，$\sigma$控制分布宽度。

本文采用自适应$\sigma$策略：

$$\sigma = \sigma_{min} + (\sigma_{max} - \sigma_{min}) \cdot \frac{y}{y_{max}}$$

年龄越大，标签不确定性越高，分布越宽，符合生理规律。

损失函数采用KL散度：

$$L_{KL} = \sum_{j} P(j|x) \log\frac{P(j|x)}{Q(j|x)}$$

其中$Q$为网络预测的分布。

### 3.6 辅助损失函数

除KL散度外，本文引入以下辅助损失：

**L1损失**：直接约束预测年龄与真实年龄的差距

$$L_{L1} = |E[Q] - y|$$

**CDF Ranking损失**：约束累积分布函数的形状，保持年龄的序数关系

$$L_{rank} = BCE(CDF_Q, CDF_P)$$

**Mean-Variance损失**：约束预测分布的方差，鼓励网络输出更确定的预测

$$L_{MV} = MSE(E[Q], y) + \lambda \cdot Var[Q]$$

总损失为：

$$L = L_{KL} + \lambda_{L1}L_{L1} + \lambda_{rank}L_{rank} + \lambda_{MV}L_{MV}$$

## 第四章 实验设置

### 4.1 数据集

本文在AFAD（Asian Face Age Dataset）数据集上进行实验。AFAD包含约16万张亚洲人脸图像，年龄范围为0-80岁，具有年龄分布不均衡的特点。

数据集采用分层采样策略划分，确保每个年龄类别在训练集、验证集、测试集中的比例一致。划分比例为80%-10%-10%。

### 4.2 实现细节

**网络架构**：骨干网络使用ImageNet预训练的MobileNetV3-Large权重。

**训练参数**：
- 优化器：AdamW
- 学习率：3×10⁻⁴
- 权重衰减：4×10⁻⁴
- 批大小：128
- 训练轮次：120
- 学习率调度：余弦退火

**数据增强**：
- 随机裁剪（scale: 0.8-1.0）
- 随机水平翻转
- 随机仿射变换（旋转±15°，平移5%，剪切5°）
- 高斯模糊
- 颜色抖动
- MixUp（α=0.5，概率0.5）
- Safe Random Erasing（概率0.1）

**训练策略**：
- 前10轮冻结骨干网络，仅训练预测头
- EMA（指数滑动平均）衰减率0.999
- 后15轮禁用数据增强进行"硬蒸馏"

### 4.3 评估指标

采用平均绝对误差（Mean Absolute Error, MAE）作为评估指标：

$$MAE = \frac{1}{N}\sum_{i=1}^{N}|y_i - \hat{y}_i|$$

### 5.5 失败案例分析

分析MAE较大的样本，发现以下问题：
1. **极端年龄**：0-5岁和70+岁样本数量少，模型学习不充分
2. **遮挡和姿态**：大角度侧脸、口罩遮挡等情况下性能下降
3. **图像质量**：模糊、过曝/欠曝图像影响特征提取

## 第六章 总结与展望

### 6.1 总结

本文针对资源受限场景下的人脸年龄估计问题，提出了FADE-Net轻量级网络。主要贡献包括：

1. 提出Pyramid Attention Injection策略，在MobileNetV3深层注入Coordinate Attention，增强空间感知能力
2. 设计纹理-语义双流融合架构，显式建模年龄的双重表征
3. 构建包含Bottleneck SPP、DLDL-v2、Mean-Variance Loss的完整训练范式
4. 在AFAD数据集上达到MAE 3.057，参数量仅4.84M

实验结果表明，FADE-Net在轻量化方面具有显著优势，为移动端部署提供了可行方案。

### 6.2 展望

本文工作仍有改进空间：

1. **跨数据集泛化**：目前仅在AFAD数据集验证，未来可在Morph、CACD等数据集测试泛化能力
2. **极端年龄处理**：可通过数据重采样或生成模型缓解年龄分布不均衡问题
3. **实时部署优化**：可进一步进行模型量化、剪枝，加速推理过程
4. **多任务学习**：可联合学习性别、表情等属性，提升特征共享效率
